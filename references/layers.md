# Memory Layers

## Volume-Based Architecture

| Layer | Capacity | Stability | Description |
|-------|----------|-----------|-------------|
| **Core** | 20 | Stable | Values, preferences, identity |
| **Cognitive** | 100 | Stable | Knowledge, skills, expertise |
| **Behavioral** | 100 | Evolving | Habits, patterns, workflows |
| **Contextual** | 200 | Dynamic | Projects, tasks, context |
| **State** | 500 | Volatile | Working memory, progress |

## Layer Selection Guide

### Core Layer (20 items)
Use for:
- User values and principles
- Personal preferences
- Identity statements
- Fixed constraints

### Cognitive Layer (100 items)
Use for:
- Technical knowledge
- Domain expertise
- Learned skills
- Research findings

### Behavioral Layer (100 items)
Use for:
- Work habits and patterns
- Preferred workflows
- Interaction styles
- Methodology references

### Contextual Layer (200 items)
Use for:
- Current project context
- Task objectives
- Team information
- Meeting summaries

### State Layer (500 items)
Use for:
- Working memory
- Todo items
- Progress tracking
- Temporary notes
